# 🧬 CYTOSURGE Landing Page

A modern and clean landing page for a biotechnology company focused on **single-cell manipulation** and **synthetic biology solutions**.

## 🚀 Features

- Responsive layout (Desktop-first)
- Clean and modern UI design
- Organized sections:
  - Hero Section
  - Applications (Cards layout)
  - Services
  - Knowledge Center
  - About Us
  - Footer with social links
- Smooth hover effects
- Font Awesome icons integration

## 🧪 Technologies Used

- HTML5
- CSS3 (Flexbox)
- Font Awesome

## 📌 Sections Overview

### 🔹 Applications
Showcases key use cases like:
- Temporal Single-Cell Profiling
- CRISPR Gene Engineering
- Mechanobiology

### 🔹 Services
Two main offerings:
- Cell Editing Services
- FluidFM Instrument

### 🔹 Knowledge Center
Includes:
- Resources
- Publications
- Blog

## 🎯 Purpose

This project was built as a front-end practice to improve layout structuring, styling, and UI design skills.
